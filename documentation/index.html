Getting Started

Introduction

Finwice is a professional and versatile Business & Finance Consulting ReactJs Template designed for corporate websites, financial advisors, consulting agencies, and business service providers.

Designed with flexibility and performance in mind, Finwice offers a collection of beautifully crafted demo layouts including 09 Homepages, tailored for corporate firms, finance consultants, investment advisors, and business portfolios. Each layout is fully responsive and optimized for clarity and conversion across all devices and screen sizes.

Whether you're launching a corporate site, a single-page consultancy landing, or a multipage financial platform, Finwice supports both multipage & onepage configurations and includes RTL Support for multilingual and right-to-left languages. It’s more than just a template — it’s a complete toolkit for professional finance websites built with React and Next.js.

The template has been thoroughly tested across modern browsers like Chrome, Firefox, and Microsoft Edge to ensure a smooth and consistent experience everywhere.

The template was designed and tested on all devices and browsers, including Firefox, Chrome, and Microsoft Edge

Run Project

1. Download and install Node.js and NPM : You can download Node.js from https://nodejs.org. NPM comes bundled with Node.js.

2. Purchase Item: After purchase from Envato you have folder itemname-package Extract i have folder itemname. You please copy file go to D:/itemname

3. Use cmd with Windows Terminal : Go go the folder D:\itemname Type command: npm install --> npm run dev

4. In your browser navigate to http://localhost:3000

5.Thats all! Now you should see your site.

For the case Visual Studio Code Open Folder Project (open Visual Studio Code) => Open Terminal (Ctrl + `) => Type command: npm i (or yarn) --> npm install (or yarn install)
